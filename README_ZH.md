## 什么是明日方舟?
失去记忆的博士请点击[此处](https://ak.hypergryph.com)

## 您可以用此MOD做什么?
作为一名在泰拉瑞亚世界醒来的博士，您可以使用各个干员的武器和道具来进行游玩。
